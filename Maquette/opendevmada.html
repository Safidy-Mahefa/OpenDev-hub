📍 MapMada — Services numériques de Madagascar

MapMada est une carte interactive open source répertoriant tous les services numériques accessibles à Madagascar. Le projet vise à offrir une ressource centralisée pour que les citoyens et développeurs puissent trouver rapidement les services disponibles dans leur région.

🚀 Démarrage rapide

git clone https://github.com/opendevmada/mapmada
cd mapmada
npm install
npm run dev

📋 Roadmap

Sprint 1 (terminé) : Architecture, modèle de données, CRUD de base. Sprint 2 (en cours) : Interface carte Leaflet, recherche géographique. Sprint 3 : API publique REST + documentation Swagger. Sprint 4 : Mode contribution citoyenne, vérification des données.

🤝 Contribuer

Cherchez les issues labelées good first issue pour commencer. Toutes les PR doivent passer une code review d'un Mentor avant merge. Lisez le CONTRIBUTING.md pour les conventions de code.

À propos d'OpenDev Mada

Une conviction simple : les développeurs malgaches ont le talent pour construire des solutions qui comptent. Il leur manque juste un espace structuré pour collaborer, apprendre et briller.

Notre mission

Construire l'écosystème tech malgache

OpenDev Mada rassemble les développeurs malgaches de tous niveaux autour de projets open source, de défis techniques et d'un système de mentorat structuré. Tout cela, gratuitement et sans condition de diplôme.

✓Rassembler les développeurs malgaches autour de projets concrets
✓Créer un écosystème open source local, visible et reconnu
✓Favoriser l'entraide, la montée en compétences et le mentorat
✓Représenter Madagascar dans la communauté tech internationale
